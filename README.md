# Шаблон для прототипа-расширения

## Что внутри?

1.  Настроенный manifest
2.  Hot reload

## Как использовать?

1.  Скачай репозиторий
2.  Зайди в browser://extensions/
3.  Перенеси папку «src»

Если хочешь использовать импорты в js, то:

```
npm i
npm run build
npm run watch // для разработки
```
